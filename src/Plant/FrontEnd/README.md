# Plant Frontend

Genesis/Governor UI for agent manufacturing pipeline.

## Key Pages

1. **Genesis Console**: Job/skill certification
2. **Agent Creation**: 7-stage pipeline visualization
3. **Agent Simulation**: Test agents before production
4. **Constitutional Audit**: L0/L1 compliance dashboard
5. **ML Model Management**: Model versions, performance

## Tech Stack

- HTML5, CSS3, Modern JavaScript
- Design: Dark theme with manufacturing/pipeline focus

See `/main/src/CP/FrontEnd/README.md` for similar structure.
