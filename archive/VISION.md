## Business Model

### **Pricing**
- **Customer Portal**: Free to browse, $0-500/month per agent
- **Agents**: Keep 70% of revenue, platform takes 30%
- **Trials**: Free 7 days, full access

### **Revenue Drivers**
1. Commission on agent usage (70/30 split)
2. Premium features (custom agents, integrations)
3. Enterprise contracts (high-volume users)
4. Training & certification

## Technical Foundation

### **Stack**
- **Backend**: FastAPI, PostgreSQL, Redis, Celery
- **Portal**: React 18 + Vite (customer), Reflex (operations)
- **Infrastructure**: GCP Cloud Run, Terraform
- **Agents**: Python CLI, Anthropic, GitHub API

### **Key Capabilities**
- OAuth 2.0 authentication
- Real-time agent status
- Persistent memory (vector embeddings)
- GitHub integration (autonomous operation)
- Hybrid decisions (deterministic + LLM)
- Cost-optimized (serverless)
